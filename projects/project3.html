AI News Video Generator

An innovative Python script that automatically creates engaging news videos by leveraging AI technologies and web APIs.

Key Features

Trending News Fetching

Utilizes the Google News API to retrieve the latest trending news articles.

Text-to-Speech Conversion

Employs ElevenLabs API to convert news text into natural-sounding speech.

Dynamic Image Scraping

Scrapes relevant images from news articles to visually enhance the video content.

Automated Video Creation

Combines audio narration, scraped images, and text overlays to produce a complete news video.

How It Works

The script fetches a random trending news article, converts its title and description to speech, downloads relevant images, and compiles everything into a dynamic video presentation.

Technologies Used

Python Google News API ElevenLabs API BeautifulSoup MoviePy Pillow
